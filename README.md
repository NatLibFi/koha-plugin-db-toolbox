# Check, Fix, Clean, Repeat: Koha Database all-in-one Toolbox (Koha Plugin)

Ultimate Koha DB maintenance toolbox to check and fix common issues, identify and correct errors, clean up old data, ensure consistency across records. Designed to be run manually and nightly.

This plugin is a set we can gather (and we're improving it further) to help librarians and system administrators within the Koha data, ensuring optimal performance and data integrity and obsolete data removal.

This tool made as a framework for subtools, each addressing specific maintenance tasks.
Every next version will bring except fixes new elements and subtools.
